How it works

The agent workbench runs Claude Code locally as a subprocess with --output-format stream-json.

Claude uses the Bash tool to SSH into the target EC2 instance and execute commands to complete the task.

{{ config.claude_bin }} -p "…" --allowedTools Bash --max-turns 30
